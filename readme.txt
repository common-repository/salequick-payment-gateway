It is easier to install the setup. Once you have installed the plugin, all you need to do is enter your SaleQuick API credentials in the plugin settings and your website will be ready to accept credit card payments.

== Installation ==

1. Sign up for a merchant account with SaleQuick.
2. Download and install SaleQuick on your WooCommerce store.
3. Enter your API details at WooCommerce > Settings > Payments > SaleQuick.
